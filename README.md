## 1.  전략 개념

- rsi14와 rsi_ma7, rsi_ma14, rsi_ma21을 이용한 전략으로
    
    **(rsi_ma_7 < 55 or rsi_ma_14 < 55 or rsi_ma_21 < 55) and rsi < 35** 일 때 매수
    
    **(rsi_ma_7 > 45 or rsi_ma_14 > 45 or rsi_ma_21 > 45) and rsi > 65** 일 때 매도
    
    매도 조건은 35%이상의 수익률이 발생할 때와 5%의 손실률이 발생할 때이다.
    

## 2. 전략 구현

- 전략 구현 플랫폼: TradingView
- 전략 코드
    
    ```python
    //@version=5
    strategy("rsi", overlay = true, default_qty_type = strategy.percent_of_equity, default_qty_value = 100, pyramiding = 2, commission_value = 0.15)
    
    start_year = 2012
    start_month = 1
    start_day = 1
    
    end_year = 2022
    end_month = 10
    end_day = 31
    
    start = timestamp("America/New_York", start_year, start_month, start_day)
    end = timestamp("America/New_York", end_year, end_month, end_day)
    
    rsi = ta.rsi(close, 14)
    
    rsi_ma_7 = ta.rma(rsi, 7)
    rsi_ma_14 = ta.rma(rsi, 14)
    rsi_ma_21 = ta.rma(rsi, 21)
    
    longstop = strategy.position_avg_price * (1 - 0.05)
    longprofit = strategy.position_avg_price * (1 + 0.35)
    
    if(time >= start and end >= time)
    
        if (rsi_ma_7 < 55 or rsi_ma_14 < 55 or rsi_ma_21 < 55) and rsi < 35
            strategy.entry("long", strategy.long)
        if (rsi_ma_7 > 45 or rsi_ma_14 > 45 or rsi_ma_21 > 45) and rsi > 65
            strategy.exit("long", stop = longstop, limit = longprofit)
    ```
